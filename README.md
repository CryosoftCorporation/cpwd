# Copy any Present working directory to clipboard

## Adding the script
1. Download the .bat script and copy the directory to which you have downloaded or stored the script.
2. Got to your enviroment path and add the directory to system variables under path.
3. Restart any cmd/powershell windows.
4. It's ready to use. 

## Usage
Just type
```
cpwd

```
from the directory you want to copy and it will copy your present working directory to the clipboard.
